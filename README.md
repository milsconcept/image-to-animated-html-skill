# Image-to-Animated-HTML AI Skill

This repository contains a specialized custom AI Agent Skill (`SKILL.md`) designed to teach AI coding assistants (like Gemini, Antigravity, or other agentic models) how to recreate static reference images as premium, interactive, animated HTML/CSS/JS applications.

---

## What is a Custom AI Skill?

A custom AI skill is a structured set of instructions, patterns, and code templates stored directly inside the codebase (under `.agents/`). 

When an agentic AI assistant is run inside this repository, it automatically discovers and loads the skill. This immediately equips the AI with advanced knowledge on how to design and build visual layout animations, handle exact duration-pacing, and perform pristine WebM video exports with transparent alpha channels.
